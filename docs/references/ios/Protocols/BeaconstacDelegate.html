BeaconstacDelegate Protocol Reference

Conforms to NSObject
Declared in Beaconstac.h

Overview

Defines the delegate methods that will be used to receive beacon-triggered events, including ranging, camping, entry, exit, and any rules that match defined criteria.

– beaconstac:rangedBeacons:

Sent to the delegate when one or more beacons are detected by the device.

- (void)beaconstac:(Beaconstac *)beaconstac rangedBeacons:(NSDictionary *)beaconsDictionary

Parameters

beaconstac

Beaconstac instance which is the sender of this message
beaconsDictionary

A dictionary with beacon ‘UUID:major:minor’ as key and MSBeacon as value.

Parameters

beaconstac

Beaconstac instance which is the sender of this message

beaconsDictionary
