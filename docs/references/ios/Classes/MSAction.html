– executeWebhookWithParams:headers:WithCompletionBlock:

This method must be called when action type is webhook. This posts data to the URL provided in Beaconstac developer portal.

- (void)executeWebhookWithParams:(NSMutableDictionary *)postParams headers:(NSMutableDictionary *)additionalHeaders WithCompletionBlock:(void ( ^ ) ( NSNumber *statusCode , NSData *data , NSError *error ))completionBlock

Parameters

postParams

A dictionary of data that you want to post to the URL. Please note that timestamp and data available in factsDictionary is automatically posted.
additionalHeaders

A dictionary of HTTP headers with values. Plesae note that HTTP headers sent from the portal are set by default
completionBlock

A block object to be executed when the request operation finishes. It takes three arguments: statusCode- HTTP status code sent by server, data- Server response after the request has been executed, error- which is a NSError object describing the failure of the request
