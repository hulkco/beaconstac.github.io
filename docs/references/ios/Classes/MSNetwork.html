Overview

+ GET:parameters:WithCompletionBlock:

Creates and runs a network operation with a ‘GET’ request

+ (void)GET:(NSString *)URLString parameters:(id)params WithCompletionBlock:(void ( ^ ) ( NSNumber *statusCode , NSDictionary *responseJson , NSError *error ))completionBlock

Parameters

URLString

The URL string used to create the request URL.
parameters

The parameters to be sent as payload to the server.
CompletionBlock

A block object to be executed when the request operation finishes successfully. It has three arguments: statusCode - The HTTP status code of the response, responseJSON - Serialized response from server, error - An NSError object describing the reason of failure, if any.
