API Overview¶

API Objects¶

This is the overview of objects in Beaconstac API. There are six endpoints that you will encounter while using the API. The end points are for Rules, Cards, Places, Webhooks, Notifications and Tags. These are the six action types that we use.

API Reference¶

Authetication is the process that identifies the username and matches password and token for a particular user. We use three authentication levels Basic Authentication, Token Authentication and Session Authentication.
